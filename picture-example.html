Optimized Images with <picture> Element

The examples below show how to use the picture element for optimal loading on both mobile and desktop. These examples will only work after you've generated the optimized images using the create-github-optimized-images.js script.

This approach delivers:

  • WebP images on supporting browsers (smaller file size)
  • Resized images on mobile devices (faster loading)
  • Original high-quality images on desktop
  • Native lazy loading for better performance

Once you've generated the optimized images, you can replace all the image tags in your index.html with picture elements like these for the best performance.
